# Shuyi Liang Personal Homepage

This repository contains the static GitHub Pages homepage for `Fightforql.github.io`.

## Preview

Open `index.html` directly, or run:

```powershell
python -m http.server 8000
```

Then visit `http://127.0.0.1:8000/`.
